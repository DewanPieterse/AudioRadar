Continuous Wave Radar

Non-Technical Implementation

This radar method is used to measure radial velocity. That means the velocity directly to, or from the radar. You are able to set the duration of the signal.

Technical Implementation

This radar method is used to measure radial velocity. That means the velocity directly to, or from the radar. You are able to set the duration of the signal as well as the frequency of the tone.

Pulse Doppler Radar

speaker_phone

Non-Technical Implementation

This radar method is used to measure the range to objects and walls. That means the distance from the radar. You should specify the maximum size of the room and the volume.

speaker_phone

Technical Implementation

This radar method is used to measure the range to objects and walls. That means the distance from the radar. You should specify the maximum size of the room, range resolution, frequency, bandwidth, number of pulses and the volume. Note that this is for advanced users only since the radar might not operate as expected given certain parameters.
